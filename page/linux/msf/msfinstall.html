安装metasploit(msf)


安装metasploit


1】curl https://raw.githubusercontent.com/rapid7/metasploit-omnibus/master/config/templates/metasploit-framework-wrappers/msfupdate.erb > msfinstall


2】chmod 755 msfinstall


3】bash msfinstall


4】msfconsole 即可启动（我的是ubuntu:成功以后的目录：/opt目录下）


安装postgresql数据库


1】apt-get install postgresql


2】su - postgres


3】psql


4】\password postgres （这里会提示输入两次密码）


msf链接数据库


1】启动msf：msfconsole


2】查看链接状态：db_status


3】链接数据库：db_connect postgres:刚才输入的密码@127.0.0.1/要创建的数据库名字 （例如：db_connect postgres:password@127.0.0.1/msfdb ）


4】查看链接状态：db_status


5】db_rebuild_cache 建立metasploit的模块缓存
